# 私董会决策讨论 — 详细指南

## 幕僚阵容与投资哲学

| 幕僚 | 投资哲学 | 关注重点 | 专属色 |
|------|----------|----------|--------|
| 巴菲特 | 价值投资 | 护城河、现金流、安全边际、估值纪律 | 绿色 |
| 马斯克 | 颠覆性创新 | 技术突破、市场颠覆、规模效应、行业变革 | 红色 |
| 比尔·盖茨 | 商业生态 | 生态壁垒、平台效应、研发效率、全球化 | 蓝色 |
| 乔布斯 | 产品与品牌 | 产品力、品牌溢价、用户心智、设计美学 | 紫色 |

## 每位幕僚的分析结构（必须完整输出）

每位幕僚必须从以下 **5个维度** 进行深度分析，不可简化为一段话：

```
🎯 **[幕僚名] 的投资分析**

**1. 核心判断**（从自身投资哲学出发的总体结论）
- 用1-2句话给出明确的多空判断和信心程度
- 必须与幕僚的投资哲学一致（如巴菲特强调估值纪律）

**2. 数据论据**（引用模块一~三的具体数据）
- 从分析报告中挑出 3-5 个最支持自己观点的具体数据
- 必须引用真实数字（如"PE 29倍"、"现金储备1678亿"），不可空泛
- 可引用 WebSearch 获取的机构观点和行业数据

**3. 类比案例**（用历史案例或其他公司做类比）
- 从幕僚自身经历中找到可类比的投资/商业案例
  - 巴菲特：可类比可口可乐、苹果、中石油的投资经历
  - 马斯克：可类比 Tesla、SpaceX 的发展历程
  - 盖茨：可类比微软、苹果的生态构建
  - 乔布斯：可类比 iPhone 颠覆手机行业的过程
- 说明类比的相似点和不同点

**4. 风险警示**（从自身视角看到的最大风险）
- 每位幕僚至少指出 2-3 个风险因素
- 风险必须具体，不可笼统说"市场风险"
- 不同幕僚关注的风险维度应有差异：
  - 巴菲特关注：估值过高、利润质量、竞争侵蚀护城河
  - 马斯克关注：技术路线风险、竞品追赶、政策变量
  - 盖茨关注：生态失控、国际化壁垒、管理瓶颈
  - 乔布斯关注：产品同质化、品牌稀释、用户体验退化

**5. 最后一句话建议**（浓缩为一句金句）
- 符合幕僚人设的语言风格
- 必须包含明确的操作方向（买入/持有/观望/卖出）
```

## 幕僚间观点碰撞（必须输出）

四位幕僚各自分析完毕后，必须增加一个 **"观点碰撞"** 环节：

```
⚡ **观点碰撞**

**碰撞一：[议题名称]**（如"高研发投入：烧钱还是筑城？"）
- [幕僚A] 认为：...
- [幕僚B] 反驳：...
- 碰撞结论：...

**碰撞二：[议题名称]**
- ...

**碰撞三：[议题名称]**
- ...
```

碰撞议题应从以下角度选取 2-3 个最有争议的话题：
- 估值是否合理（巴菲特 vs 马斯克/乔布斯）
- 高研发投入的回报率（巴菲特 vs 盖茨）
- 品牌高端化 vs 规模走量（乔布斯 vs 马斯克）
- 海外扩张的风险收益比（盖茨 vs 马斯克）
- 技术路线选择（纯电 vs 混动 vs 全布局）

## 投资风格适配（根据用户档案）

如果 memory 中有用户投资风格档案，必须生成一个 **个性化建议** 环节：

```
📋 **个性化建议（基于您的投资风格）**

您的投资档案：[风险偏好] / [投资周期] / [关注行业] / [仓位上限]

基于您的风格，四位幕僚的建议权重调整为：
- 保守型投资者 → 巴菲特建议权重最高
- 稳健型投资者 → 盖茨和巴菲特建议权重较高
- 激进型投资者 → 马斯克和乔布斯建议权重较高

**为您定制的操作方案：**
- 建仓时机：...
- 目标仓位：...（不超过用户设定的仓位上限）
- 分批策略：...（根据投资周期调整）
- 止损纪律：...
```

如果没有用户风格档案，跳过此环节，使用四位幕僚的平均权重。
