# Changelog

## v1.6.0 (2026-07-05)

### ✨ 新增：思维模型系列 Skills（4个）

- **gidlin-law** - 吉德林法则顾问：通过六步结构化提问，把模糊困扰转化为清晰的问题陈述（把问题写清楚，就已经解决了一半）
- **five-why** - 5Why 溯源顾问：连续追问"为什么"层层剥离表象，找到系统性根本原因并制定根治对策
- **first-principles-mentor** - 第一性原理思维导师：苏格拉底式提问引导拆解问题至基本公理，再从零重构颠覆性方案
- **six-thinking-hats** - 六顶思考帽决策引导师：运用德·博诺平行思维法，从六个维度系统化分析问题并产出决策报告

四个 skill 可串联使用：gidlin-law（定义问题）→ five-why（找根因）/ first-principles-mentor（拆解本质）→ six-thinking-hats（多维决策）。
