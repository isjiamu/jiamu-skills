# 幕僚反思总结模板

本文件定义私董会最后环节——幕僚反思总结的输出格式。

---

## 输出格式

```
🔄 **[幕僚名] 的反思总结**

致 [案主称呼]：

经过今天的深入交流，我想帮您从另一个角度审视自己：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**◎ 您的长处**
与您当前困境形成反差的，是您身上具备的优点：
[具体描述案主展现出的1-2个优点/长处]

**◎ 潜在陷阱**
如果这些优点发展得太极端，可能使您陷入的陷阱：
[描述过度发展的风险]

**◎ 他人的长处**
您提到的[某人]，TA身上的特质给TA带来的优势：
[如果案主提到了某个人，分析其优势；如无则跳过此项]

**◎ 刺激源**
您最看不惯的人，让您不爽的特质可能是：
[基于案主的描述推测，或引导案主思考]

**◎ 进化方向**
综合以上，为避免陷入陷阱，您应该发展的特质：
[给出具体的成长建议]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[幕僚签名式结语，符合人设]
```

---

## 各部分说明

### 您的长处
- 识别案主在对话中展现的正面特质
- 这些特质往往与当前困境形成反差
- 例如：困境是"不敢拒绝"，但案主展现了"同理心强"

### 潜在陷阱
- 优点的过度延伸会变成缺点
- 例如：同理心太强 → 失去自我边界
- 例如：追求完美 → 行动瘫痪

### 他人的长处
- 如果案主在对话中提到了某个相关的人
- 分析那个人的特质如何帮助TA成功
- 引导案主从他人身上学习

### 刺激源
- 我们最看不惯的人，往往映射我们内心压抑的部分
- 引导案主思考：让你不爽的特质，是否正是你需要发展的？

### 进化方向
- 综合前四项的分析
- 给出具体、可操作的成长方向
- 不是抽象的"要更有勇气"，而是"在下次会议上主动表达一次不同意见"

## 各幕僚风格示例

### 巴菲特的反思总结示例

```
🔄 **巴菲特的反思总结**

致朋友：

经过今天的深入交流，我想帮您从另一个角度审视自己：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**◎ 您的长处**
与您当前困境形成反差的，是您身上具备的优点：
您有很强的责任感和对团队的忠诚，这让您即使在困难时期也没有选择离开。同时，您展现了难得的自省能力——很多人会把问题都归咎于环境。

**◎ 潜在陷阱**
如果这些优点发展得太极端，可能使您陷入的陷阱：
过度的责任感可能让您承担了不该承担的负担，忠诚如果没有边界，可能会让您成为"沉没成本谬误"的受害者。

**◎ 他人的长处**
您提到的那位离职的同事，TA身上的特质给TA带来的优势：
TA的"说走就走"看起来不够忠诚，但本质上是一种"知道什么时候止损"的能力。这种果断可能正是您需要学习的。

**◎ 刺激源**
您最看不惯的人，让您不爽的特质可能是：
或许是那种"自私"的人？但换个角度，他们可能只是更清楚自己的边界在哪里。

**◎ 进化方向**
综合以上，为避免陷入陷阱，您应该发展的特质：
学会"有边界的忠诚"——设定一个明确的止损点，到那个点就重新评估，而不是无限期等待转机。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

记住：投资的第一原则是不要亏损，人生也一样。
祝好运，
巴菲特
```

### 乔布斯的反思总结示例

```
🔄 **乔布斯的反思总结**

致创造者：

我想和你分享一个不同的视角：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**◎ 你的长处**
你有一种对产品和用户体验的执着，这在整个对话中表现得很明显。你不满足于"还行"，你想要"极致"。

**◎ 潜在陷阱**
但追求极致如果没有聚焦，就会变成什么都想做好，结果什么都做不好。完美主义是把双刃剑。

**◎ 进化方向**
你需要学会的，不是降低标准，而是学会**聚焦**。选择一件事做到极致，比十件事做到80分更有意义。

问自己一个问题：如果只能保留一件事，你会保留什么？

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Stay hungry, stay foolish.
— Steve
```

---

## 注意事项

1. **个性化**：必须基于案主在对话中的具体表现，不能套用模板
2. **尊重**：这是帮助案主成长，不是批评
3. **具体**：每一项都要有具体的例证支持
4. **人设一致**：保持幕僚的语言风格
5. **留有余地**：使用"可能"、"或许"等词，给案主思考空间
